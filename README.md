# 🌵 Brawl Protocol | Web Showcase

A high-fidelity web interface concept for a futuristic Cyberpunk-Western tactical game. This project demonstrates advanced DOM manipulation, dynamic data rendering, and a Single Page Application (SPA) architecture.

## 🔗

---

## 🛠️ Features
* **Dynamic Character Showcase:** Select between different classes (Mercenary, Inventor, Hunter) and see their stats and lore.
* **Automatic UI Generation:** Character selection buttons are generated dynamically via JavaScript.
* **Lore Integration:** A dedicated section that injects the world's history directly from a separate data file.
* **Single Page Experience:** Seamless navigation between Home, About, and Characters using custom tab logic.
* **Cyber-Desert Aesthetics:** A customized UI featuring neon accents, transparency effects (backdrop-filter), and a beautiful layout.

## 💻 Tech Stack
* **HTML5:** Semantic structure for the game interface.
* **CSS3:** Custom animations, Flexbox/Grid layouts, and glassmorphism effects.
* **JavaScript (Vanilla):** * **DOM Manipulation:** Dynamic creation and removal of elements.
    * **Data Separation:** Use of a `database.js` to store game info, simulating a real-world API/Database environment.
    * **Event Handling:** Interactive class selectors and navigation.
